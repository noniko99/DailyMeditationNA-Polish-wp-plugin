# Daily Meditation NA - Polish (WordPress plugin).

Wtyczka WordPress wyświetlająca codzienną medytację w języku polskim na podstawie danych zapisanych w bazie danych.

![WordPress](https://img.shields.io/badge/WordPress-Plugin-blue)
![Version](https://img.shields.io/badge/version-1.0.4-green)

---

## Informacje

**Nazwa:** Daily Meditation NA - Polish  
**Wersja:** 1.0.4  

Wtyczka została stworzona do automatycznego wyświetlania codziennej medytacji.

Po aktywacji:
- tworzona jest wymagana tabela w bazie danych WordPress,
- importowane są dane z pliku SQL,
- shortcode pozwala wyświetlić aktualną medytację na stronie.

---

# Funkcje

✅ Codzienna medytacja według daty  
✅ Automatyczne tworzenie tabeli w bazie danych  
✅ Import danych SQL podczas aktywacji  
✅ Prosty shortcode WordPress  
✅ Panel informacji w administracji WordPress  
✅ System aktualizacji z GitHub
